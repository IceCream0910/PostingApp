지금까지 배운 변수만으로는 여러개의 데이터 집합을 표현하기에 어려움이 있다.
그래서 배열이라는 것이 존재한다.
예를 들어 1부터 5까지의 정수 집합은 다음과 같이 int 배열로 표현할 수 있다.
int[] array = {1, 2, 3, 4, 5};
배열은 변수타입 바로 오른쪽에 []기호로 표현한다.
마찬가지로 문자열 배열도 다음과 같이 표현할 수 있다.
String[] weeks = {"월", "화", "수", "목", "금", "토", "일"};
즉, 배열이란 여러개의 데이터를 변수에 넣는 것이다.
배열은 다음과 같이 사용할 수도 있다.
String[] weeks = new String[7];
weeks[0] = "월";
weeks[1] = "화";
weeks[2] = "수";
weeks[3] = "목";
weeks[4] = "금";
weeks[5] = "토";
weeks[6] = "일";
이렇게 배열의 길이만 먼저 선언한 후 그 값은 그 다음에 대입하는 방법이다. 단 이렇게 초기값 없이 배열을 만들 때는 길이값이 꼭 필요하다.
String[] weeks = new String[];
이렇게 하면 안된다는 것이다.
그렇다면 배열의 값에는 어떻게 접근할 수 있을까?
String[] weeks = {"월", "화", "수", "목", "금", "토", "일"};
System.out.println(weeks[3]);
이렇게 변수명[값위치] 로 배열에 있는 데이터를 가져올 수 있다.
여기서 주의해야 할 것은 java에서는 숫자를 0부터 세기 때문에, weeks[1]은 두번째 값인 "화"를 의미한다.
배열을 사용하다보면 배열의 길이를 알아야 할때가 있다.
그럴 때는 변수명.length를 사용하면 된다. 이렇게.
String[] weeks = {"월", "화", "수", "목", "금", "토", "일"};
for (int i=0; i System.out.print(weeks[i]);
}
